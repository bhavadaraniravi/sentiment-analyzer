# Sentiment Analyzer Pro

An AI-powered web application that detects emotions and sentiment in any text using Natural Language Processing (NLP).

## Live Demo
Try it here: https://huggingface.co/spaces/daranirbhava/sentiment-analyzer

## Features
- Sentence by sentence sentiment analysis
- 5 level sentiment detection (Very Positive to Very Negative)
- Emotion detection (Joy, Anger, Sadness, Fear, Surprise)
- Word cloud generation
- Bar chart and pie chart visualization
- Bulk CSV file analysis
- Permanently hosted on Hugging Face Spaces

## Tech Stack
- Python
- TextBlob (NLP)
- Gradio (UI)
- Matplotlib (Charts)
- WordCloud
- Hugging Face Spaces (Deployment)

## How to Run Locally
1. Clone this repository
2. Install requirements:
pip install -r requirements.txt
3. Run the app:
python app.py
4. Open the local URL shown in terminal
